# Worksheet Phase Notes

## Current Phase

This phase is limited to trustworthy teacher-provided homework playback:

- route likely worksheet assets into homework storage
- preserve original PDF/image assets
- prefer PDF overlay playback when trusted assets exist
- keep assignment truth authoritative for grading and resume
- log clearly when the system uses the PDF player versus SVG fallback

## Explicitly Deferred

The following are intentionally not part of this phase:

- Sunny-authored exercise templates
- visual polish work for generic SVG worksheet scenes
- broad removal of SVG from non-worksheet flows
- automatic overlay detection from OCR or PDF layout

## Why

Parents need confidence that homework sessions are showing the real worksheet and grading against server-owned truth. Template visuals are a separate product step after the PDF overlay path is dependable.
