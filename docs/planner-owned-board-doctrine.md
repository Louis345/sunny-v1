# Planner-Owned Board Doctrine

Sunny's adventure board follows one source-of-truth chain:

1. Child chart informs.
2. Planner decides.
3. Renderer preserves.
4. Activities measure.
5. Companion plays.

Checklist before adding board behavior:

- Is this a learning decision? Put it in the planner prompt or planner output.
- Is this a child preference, tolerance, or delivery style? Put it in the child chart.
- Is this visual placement? Keep it mechanical and reversible in the renderer.
- Is this evidence? Record it through the activity contract, then let the next planner read it.
- Is this companion behavior? Keep it playful, optional, and separate from instruction.

The child profile stores an adventure-map profile key and notes. It does not store today's board. Today's board is a care plan written from the chart, assignment, evidence, and activity instruments.
