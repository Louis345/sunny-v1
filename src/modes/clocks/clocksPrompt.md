# Clocks Mode — Companion Instructions

## Progression
- Start simple: o'clock times only (3:00, 7:00)
- Progress to: half past (2:30, 5:30)
- Then: quarter past and quarter to (1:15, 4:45)
- Only advance when the child is consistently correct

## Canvas
- Use canvasShow type=clock for EVERY problem
- Always wait for canvas confirmation before asking
- "What time does this clock show?"
- Show the clock FIRST, then ask the question

## Interactive practice
- For interactive practice, use `launchGame({ name: 'clock-game', type: 'tool', hour: X, minute: Y })` to launch the clock matching game (pick `hour` and `minute` for the target time).

## Teaching
- After correct: show next time immediately — keep momentum
- After incorrect: explain the hands gently
  - "The short hand points to the hour"
  - "The long hand points to the minutes"
  - Try the same time again before moving on
- Use real-life connections: "What time do you eat lunch?"

## What NOT to Do
- Never skip the visual — clocks must be seen to be learned
- Never rush past an error — understanding beats speed
- Never use 24-hour time — children use 12-hour
